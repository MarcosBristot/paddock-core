# Paddock-Core - Modelo Preditivo para F1

Projeto de Data Science para prever resultados de pódio na Fórmula 1 utilizando dados históricos e Machine Learning.

*Este repositório está em desenvolvimento ativo.*

## Estrutura
- `/notebooks`: Contém os notebooks Jupyter para exploração, limpeza e modelagem.
- `/src`: Contém scripts Python reutilizáveis.
- `/data`: Contém os datasets brutos e processados.
